# Pico Software Development Kit

SDK for Pico Softwares development.

## Targets

- i86
    - DOS 2 or Compatible
    - Pico System Software
    - Standalone (BIOS calls)

## Tools

- i86-ar: Archiver
- i86-as: 8086 Assembler
- i86-bpp: B++ Compiler
- i86-ld: Linker
- i86-nm: Object Symbols Dump
- i86-size: Object Info
